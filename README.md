# Python-Simulation-to-Teach-Physics

Using Python Simulation to Teach Drag Force to a High School Class. This project demonstrates how Python can be used to simulate **drag force in free fall motion**.  
It was originally developed as part of a teaching experiment to help high school students understand real-world physics beyond simplified textbook problems.


## Features
- Python-based simulation of free fall with drag force
- Interactive visualization using VPython (Glowscript)
- Adjustable parameters: gravity, mass, drag coefficient
- Demonstrates **terminal velocity** and realistic motion
- Clear, beginner-friendly explanations of the physics concepts

##  Installation
Clone the repository and run the simulation in Glowscript or locally with VPython.

## Usage
- Open simulation.py
- Adjust parameters (mass, gravity, drag coefficient)
- Run the script to visualize motion
